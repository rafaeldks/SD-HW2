Работа выполнялась на Mac OS, из-за чего в качестве символов разделителей в путях в коде используется "/".

При запуске программы нужно ввести абсолютный путь до корневой папки, после чего будет автоматически обработано её содержимое и построен список зависимостей в файлах (каждая компонента связности будет выведена через новую строку)

При наличии циклов в зависимостях в файлах программа завершается с кодом возврата 1.

В самих файлах "require" должен начинаться с новой строки.

После вывода списка зависимостей содержимое всех файлов в этом списке будет склеено в том же порядке в один текстовый файл "Result.txt", находящийся в папке с Idea проектом.
